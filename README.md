##  Description

Site web professionnel pour TechSolutions Pro, une entreprise spécialisée dans les solutions informatiques innovantes. Le site présente les services de l'entreprise en développement web, applications mobiles et solutions cloud.

##  Fonctionnalités

- **Navigation responsive** avec menu hamburger pour mobile
- **Section Hero** avec animations et cartes flottantes
- **Présentation des services** avec grille de cartes interactives
- **Statistiques animées** qui s'activent au scroll
- **Témoignages clients** avec design moderne
- **Footer complet** avec liens sociaux et informations de contact
- **Animations fluides** et effets de survol
- **Design responsive** adapté à tous les écrans

##  Technologies Utilisées

- **HTML5** - Structure sémantique
- **CSS3** - Styles modernes avec variables CSS, Grid et Flexbox
- **JavaScript Vanilla** - Interactions et animations
- **Font Awesome** - Bibliothèque d'icônes

##  Installation et Utilisation

1. Cloner le repository
```bash
git clone [URL_DU_REPO]
```

2. Ouvrir le fichier `index.html` dans votre navigateur

Aucune installation de dépendances n'est nécessaire, le site utilise uniquement des technologies web natives.

##  Fonctionnalités JavaScript

- **Navigation mobile** : Toggle du menu hamburger
- **Animation des statistiques** : Compteurs animés au scroll
- **Apparition des cartes** : Effet fade-in au scroll avec IntersectionObserver
- **Smooth scroll** : Navigation fluide vers les sections

##  Responsive Design

Le site est entièrement responsive avec des breakpoints à :
- **968px** : Adaptation pour tablettes
- **768px** : Optimisation mobile complète
